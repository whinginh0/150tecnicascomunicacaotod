Crie uma página de vendas completa no Antigravity para o produto digital:

A página deve ser 100% MOBILE FIRST, feita exclusivamente para público mobile. Não precisa otimizar para desktop ou PC. Toda a estrutura deve ser pensada para tela de celular, com largura confortável, leitura fácil, botões grandes, elementos centralizados e navegação simples.

Não usar visual poluído.

PALETA DE CORES
==================================================

A paleta deve reunir as cores dos entregáveis, mas todas em versões claras, suaves e clean.

Usar:

- Fundo principal: branco ou off-white;
- Textos principais: preto suave ou grafite escuro;
- Roxo claro: para identidade principal da página;
- Amarelo claro: para detalhes, destaques leves e elementos de atenção;
- Vermelho claro: para pequenos detalhes de urgência e faixa de oferta;
- Verde claro: para bônus grátis, garantia e segurança;
- Azul claro: para elementos de confiança, cards e destaques;

As cores não podem ser fortes, neon ou agressivas.
Usar tons pastel, suaves, elegantes e educativos.
